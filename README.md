# 🚀  AI Image Generator

An AI Image Generator is a full-stack AI-powered image generation platform built using the MERN stack.  
Users can generate stunning images using Google's Gemini 2.5 Flash Image model with secure authentication and daily generation limits.

---

## 🌟 Features

- 🔐 JWT Authentication (Login / Signup)
- 🎨 AI Image Generation using Gemini 2.5 Flash Image
- 🖼 Image Download Support
- 📅 Daily Limit System (3 images per user & per IP)
- 🧠 MongoDB Image Storage
- 📱 Fully Responsive UI
- 🌙 Dark Mode Support
- ⚡ Fast Vite + React Frontend

---

## 🛠 Tech Stack

### Frontend
- React (Vite + TypeScript)
- Tailwind CSS
- shadcn-ui
- Lucide Icons

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Google Gemini API
